# Figures & Tables 

This folder contains scripts and outputs used to generate the figures and tables for the manuscript



## Contents

| File | Description |
|---|---|
| `Manhattan_plot` | Scripts and/or outputs for generating Manhattan plots of GWAS results |
| `Table1.R` | R script used to generate Table 1 summary statistics |
| `circle_plot_code.R` | R script for generating circular/circos-style visualization plots |
