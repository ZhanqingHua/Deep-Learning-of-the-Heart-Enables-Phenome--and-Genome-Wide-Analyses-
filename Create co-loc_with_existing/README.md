# Cardiac Radiomics GWAS

This repository contains scripts and files for overlap analysis, LD clumping, annotation, and colocalization analyses used in the cardiac radiomics GWAS project.

## Repository Structure

### `overlap&clumping&annotation/`
Contains scripts and intermediate files for:
- overlap analysis between radiomic GWAS and published cardiovascular disease GWAS
- LD clumping
- nearest gene annotation
- downstream functional interpretation

### `Co-loc/`
Contains scripts for colocalization analyses between radiomic GWAS loci and cardiovascular disease GWAS signals.

## Notes
- Additional analyses and scripts will be continuously updated.
- Some files are omitted due to data-sharing restrictions and file size limitations.
